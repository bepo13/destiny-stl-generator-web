Destiny STL Generator

Generate STL files for thousands of weapons, armor and ships from the Destiny universe. Models are created with the mobile assets used on the official Destiny mobile app and Bungie website.

  1. Select an item from below and click generate.
  2. Download generated STL file or ZIP file containing all parts.
  3. Import files into your favorite CAD program or clean up using an online service such as Microsoft 3D Tools.
